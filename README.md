# JEAF Persistence Service Provider Base README #

This repository contains the source code of JEAF Persistence Base. The project contains generic base classes that can be used to implement a JEAF Persistence Service Provider 

## Links ##
For further information please refer to:

* [JEAF Persistence](https://anaptecs.atlassian.net/l/c/E5zcD4xT)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.persistence/jeaf-persistence-service-provider-base)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.persistence/jeaf-persistence-service-provider-base/latest/index.html)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
